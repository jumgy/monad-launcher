# monad-launcher

Установка и использование
Скачиваем и помещаем в папку с софтом

Открываем консоль, переходим в папку с софтом:

bash
cd ваша\папка\с\софтом
Запускаем лаунчер (когда на всех акках есть баланс):

bash
python launcher.py
Настраиваем:

Конфиг
Модули
Расписание (ставьте количество часов с запасом)
Запускаем софт
